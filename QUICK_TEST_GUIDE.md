### 第2步: 测试新插件 - 添加水印 (5分钟)

1. 点击"添加水印"插件卡片
2. 上传一个测试Excel文件
3. 配置水印:
   - 选择"文本水印"
   - 输入水印文字: "TEST"
   - 调整字体大小: 48
   - 选择颜色: 灰色
   - 设置透明度: 128
   - 旋转角度: -45°
4. 点击"下一步"
5. 点击"开始处理"
6. 等待处理完成
7. 下载处理后的文件
8. 打开文件验证水印是否添加成功

**预期结果**: 文件成功添加水印，可以下载

## 🐛 常见问题排查

### 问题1: 页面无法加载
**解决方案**:
```bash
# 检查开发服务器是否运行
# 如果没有运行，执行:
npm run dev
```

### 问题2: 插件显示空白
**检查**:
- 浏览器控制台是否有错误
- 插件文件是否存在
- 路由配置是否正确

### 问题3: Python脚本执行失败
**检查**:
- Pyodide是否加载成功
- Python依赖包是否安装
- 文件格式是否正确

### 问题4: 文件下载失败
**检查**:
- 浏览器是否阻止下载
- 文件大小是否超限
- 处理结果是否包含data字段
